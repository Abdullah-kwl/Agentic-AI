# 🤖 Agentic AI Revision Repo

This repository is focused on **revising and understanding Agentic AI concepts** using modern frameworks like **LangChain** and **LangGraph**.

---

## 📌 Purpose

This repo is **NOT for production-level projects**.

Instead, it is designed for:

* 🧠 Revising core Agentic AI concepts
* 🧩 Understanding how agents work step-by-step
* ⚡ Quick hands-on experiments
* 📚 Building strong conceptual clarity

Think of this as your **personal revision notebook + mini lab**.

---

## 🚀 What You’ll Learn

* What is Agentic AI
* How AI Agents work
* Tool usage in agents
* Memory & context handling
* Multi-step reasoning
* Workflows using LangGraph
* Basic RAG + Agents integration

---

## 🛠 Tech Stack

* Python 🐍
* LangChain
* LangGraph
* OpenAI APIs

## ⚡ How to Use

1. Clone the repo

```bash
git clone <your-repo-link>
cd <repo-name>
```

2. Create environment (recommended using uv)

```bash
uv venv
.venv\Scripts\activate
```

3. Install dependencies

```bash
uv pip install -r requirements.txt
```

4. Run examples

```bash
python basics/example.py
```
